Overview

The Visitor Management System (VMS) is comprised of a web-based scheduling app, iPad app, and name badge printer. Visitors receive a generated name badge with a QR code that allows them to check-in and check-out with a touchless experience.

Aqua employees schedule, send email invitations, receive email notifications when their visitor arrives, and can view multiple reports.

Have questions? Contact Chuck Konkol (ext. 4574) or email ckonkol@aqua-aerobic.com.

1. Requirements

To ensure the system works correctly, please ensure the following:

  • Use Chrome Web Browser.
  • Set your default mail app to Outlook.
  • Watch the step-by-step video.

2. Schedule a Visitor

  1. Access the Scheduler: Go to aquavisitorsystem.github.io and click on Schedule Visitor.
  2. Enter Details: Fill in the required fields.
    Unknown Guest Name? Enter TBD in the name fields. Guests can update their data upon arrival.

    Multiple Dates: You can add up to 12 dates per visitor by clicking [Click to add/view more dates].

  3. Save: Click the Schedule Visitor button.
  4. Send Invitation (Optional): Click "Click here to create email to guest". This generates an email with the QR code.

Weekly Recurring Dates

  1. Enter required guest fields.
  2. Remove any individual dates if applicable.
  3. Select Weekly Recurring Dates.
  4. Select Start Date, End Date, Time, and Days of the week.
  5. Click Schedule Guest.

View/Edit Visitor Schedule

To edit an existing visit:

  1. Go to the website and click View Scheduled Visits.
  2. Enter your Aqua User ID (e.g., ckonkol).
  3. Click the Click here link next to the schedule you want to edit.
  4. Update the fields and click Update Visit.

3. Update Visitor Schedule: Active/In-Active

By default, schedules are active. If a visit is cancelled, you should mark it as In-Active.

  1. Search for your schedule using your User ID.
  2. Select the schedule to edit.
  3. Click the link below Status to toggle between Active and In-Active.
Note: In-Active schedules are removed periodically. Contact IT if you need to restore a deleted schedule.

4. Check-In/Check-Out Using iPad App

Option A: QR Code

Show your QR code to the iPad camera to scan instantly.

Option B: Manual Tap

  1. Tap the green button on the iPad screen.
  2. Find your name/schedule and tap the HERE link.
  3. Update any details if necessary.
  4. Tap Tap Here to Check-in/Out.
Checking Out? Upon check-out, please dispose of your name badge. The host will receive an email notification.

7. Reports

Go to View Scheduled Visits and enter a keyword or User ID in the search box.

Keyword Description
[Aqua User ID] Shows active schedules for that user (e.g., ckonkol).
today Show today's schedule.
myvms Shows active schedules for your User ID.
activitylog Shows live visitor log.
active Shows all schedules greater or equal to today's date.
name Search by guest last name.
company Search by guest company name.
logall Get report of ALL check-in/check-out sessions.
printjobs Report of label usage and remaining labels.

8. Reset Check-in/Check-out Data

Use this if the printer failed or there was a power issue and the status is incorrect.

  1. Go to View Scheduled Visits.
  2. Search for today.
  3. Select the specific guest.
  4. Click the link: Click here to reset check-in/check-out data.

Update App

  1. Close AquaCheck-In VMS App (See "Restart iPad" below for closing instructions).
  2. Open Safari and go to: aquavisitorsystem.github.io/install.html
  3. Tap Get Update! then Install.
  4. Go to Settings > General > VPN & Device Management > Aqua-Aerobic Systems, Inc. and tap Trust.

Setup & Guided Access

  • Wi-Fi: Ensure iPad is connected to printer Wi-Fi: DIRECT-06442_QL-1110NWB. (Internet comes from Cellular).
  • iPad/App Code: 6306
  • Start Guided Access: Triple tap the top-left button.
  • Exit Guided Access: Triple tap top-left button and enter code 6306.

Restart iPad / Close App

To close the app, perform a five-finger pinch on the screen.
