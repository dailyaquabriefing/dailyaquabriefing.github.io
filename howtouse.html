How to Use Your AquaDailyBriefing

Here is your simple, two-step daily process to get your briefing.

1. Update Your Files (If Needed)

Before you run the app, you can quickly make any changes to:

  • Your projects and tasks: G:\Active_Projects.xlsx
  • Your recurring daily goals: G:\DailyTask.txt

This step is optional. If everything is current, just skip to step 2.

2. Run the App

Just double-click the application: G:\AquaDailyBriefing.exe

What Happens Automatically

When you run the app, it instantly does all the gathering work for you:

  • Gets daily tasks from G:\DailyTask.txt.
  • Gets active projects from G:\Active_Projects.xlsx.
  • Gets unread emails from Outlook (from the last 24 hours).
  • Gets your meetings (from Today through Friday) from Outlook.
  • Pushes all this fresh data to your briefing website.

After a few seconds, the app will confirm success and ask if you want to view your briefing!

Pro-Tips for Automation

Make your daily sync even easier with these optional steps.

Create a Desktop Shortcut

For fast manual syncing, create a shortcut to the app right on your desktop:

  1. Navigate to G:\ in File Explorer.
  2. Right-click on AquaDailyBriefing.exe.
  3. Select Send to > Desktop (create shortcut).

Now you can run the sync anytime without opening the G: drive.

Run Automatically with Task Scheduler

Set Windows to run the sync for you automatically, so your briefing is always up-to-date during the day.

  1. Open the Task Scheduler app (search for it in the Start Menu).
  2. In the "Actions" pane, click Create Basic Task....
  3. Follow the wizard:
    • Trigger: Select "Daily" and set a start time, like 7:00 AM.
    • Action: Select "Start a program" and browse to G:\AquaDailyBriefing.exe.
  4. After creating the task, find it in the "Task Scheduler Library", right-click it, and go to Properties.
  5. Go to the Triggers tab and click Edit....
  6. Under "Advanced settings", check Repeat task every: and set it to 1 hour (or your preferred frequency).
  7. Set for a duration of: to 10 hours (to cover 7 AM to 5 PM).

This will automatically refresh your data all day without you having to do anything!
