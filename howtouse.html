How to Use Aqua Daily Briefing

üëã Say goodbye to juggling multiple apps! The Aqua Daily Briefing is your personalized daily co-pilot. It combines your Task Priorities with automated Outlook Data into one secure, shareable dashboard.

The Two-Part System

  1. The Web Dashboard: Your command center for managing Tasks, Projects, and Security Settings.
  2. The Outlook Agent: A background tool that securely syncs your calendar and emails to the report.

Step 1: Admin Dashboard & Setup

Use the Admin Dashboard to manage what appears on your report.

  1. Go to the Admin Dashboard.
  2. First Time? Click "First Time User? Setup Here" on the login screen to link your email to your Aqua User ID (e.g., jdoe).
  3. Logging In: Use your email and password to access the dashboard.

Step 2: Managing Your Content

The Admin Dashboard is organized into four tabs. Content supports Rich Text (Bold, Bullets, Headers) via the new editor.

1. Daily Tasks Tab

  • Add your top 3-5 focus items for the day.
  • Use the Up/Down buttons to prioritize them.

2. Projects & Active Tasks Tabs

These sections support rich data. When adding or editing an item:

  • Rich Text Notes NEW: Use the editor to format your notes with bullet points, bold text, and headers.
  • Multiple Attachments NEW: You can now upload multiple files or add manual links (e.g., SharePoint URLs) to a single task.
  • Collaborators: Tag teams (e.g., "Design Team") to display a yellow Team badge.
  • Milestones: Track the next major goal (üèÅ).
  • Dates: Set Start and End dates. The Last Updated field updates automatically.
  • Private Comments NEW: Private notes for internal use.

3. Settings Tab (Security)

  • Passcode Protection: Set a passcode here to lock your Daily Briefing view.
  • Important: You must set a passcode to enable the "Private Comments" field.

Step 5: Public & Private Comments

Stakeholders can ask questions, and you can keep private notes secure.

  • Public Comments: Viewers can click "üí¨ View/Add Comments" to chat. The new editor supports Rich Text formatting.
  • Private Comments: Available in the Admin Edit modal. These notes are hidden from the public and only visible when the Report Passcode is entered.

Step 6: Export to Excel IMPROVED

Need a hard copy? The export feature has been upgraded.

  • Where: Click 📥 Export to Excel in the top nav.
  • What's New:
    • Includes the full Daily Check History log.
    • Includes Analytics Overview as the first sheet.
    • Automatically strips HTML for clean, readable text cells.
    • Adjusts column widths automatically for readability.
  • Smart Security:
    • If exported from the Public View, it excludes Private Comments and Outlook Data.
    • If exported from the Private (Unlocked) View, it includes Private Comments, Meetings, and Emails.
