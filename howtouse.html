How to Use Aqua Daily Briefing

üëã Say goodbye to juggling multiple apps! The Aqua Daily Briefing is your personalized daily co-pilot. It combines your Task Priorities with automated Outlook Data (emails/meetings) into one secure, shareable dashboard.

The Two-Part System

  1. The Web Dashboard: Your command center for managing Tasks, Projects, and Security Settings.
  2. The Outlook Agent: A background tool that securely syncs your calendar and emails to the report.

Step 1: Admin Dashboard & Setup

Use the Admin Dashboard to manage what appears on your report.

  1. Go to the Admin Dashboard.
  2. First Time? Click "First Time User? Setup Here" on the login screen to link your email to your Aqua User ID (e.g., jdoe).
  3. Logging In: Use your email and password to access the dashboard.

Step 2: Managing Your Content

The Admin Dashboard is organized into four tabs:

1. Daily Tasks Tab

  • Add your top 3-5 focus items for the day.
  • Use the Up/Down buttons to prioritize them.

2. Projects & Active Tasks Tabs

These sections support Rich Data. When adding or editing an item, you can specify:

  • Status: Color-coded badges (e.g., On Track, Delayed).
  • Resources NEW: Upload files or paste links to Drive/SharePoint.
  • Milestones: Track the next major goal (üèÅ).
  • Dates: Set Start, End, and Last Updated dates.
  • IT Comments NEW: Private notes for the IT team.
  • Public Comments NEW: View and delete public comments directly from the edit window.

3. Settings Tab (Security)

  • Passcode Protection: Set a passcode here to lock your Daily Briefing view.
  • Important: You must set a passcode to enable the "IT Team Only Comments" field in the editor.

Step 3: Private IT Comments NEW

You can now add private notes that are hidden from the public.

  • How to Enable: Go to Settings and save a Passcode.
  • How to Add: In the Admin modal (Edit/Add Item), a new red field labeled "üîí IT Team Only Comments" will appear.
  • Visibility: These notes ONLY appear in the Daily Briefing view after you successfully enter the passcode. They are completely removed from the Weekly Report.

Step 4: Public Comments & Collaboration NEW

Stakeholders can now ask questions or leave feedback directly on your report.

  • How to Comment: Viewers can click "üí¨ View/Add Comments" on any task, enter their Name, and click Post.
  • Where it Appears: Comments are visible on both the Public Weekly Report and the Private Daily Briefing.
  • Moderation: To delete an inappropriate comment:
    1. Log in to the Admin Dashboard.
    2. Click Edit on the specific task.
    3. Scroll down to the "üí¨ Public Comments" section.
    4. Click the Delete button next to the comment you wish to remove.

Step 5: File Uploads NEW

You can now host documents directly within your report (e.g., Specs, PDFs, Screenshots).

  • How to Upload: In the Admin "Edit Item" window, find the Resource section.
  • Click "Upload": Select a file from your computer (PDF, Word, Excel, Text, or Image).
  • Limits: The maximum file size is 10MB.
  • Automatic Linking: Once the upload finishes, the system automatically inserts the secure link for you.

Step 6: Export to Excel NEW

Need a hard copy or backup? You can now export your data to Excel.

  • Where: Click the 📥 Export to Excel link in the top navigation bar.
  • What is Exported: All Tasks, Projects, and Active Tasks.
  • Analytics Included: The export now includes a summary Analytics Overview sheet as the first tab.
  • Smart Security:
    • If exported from the Public View, it excludes IT Comments and Outlook Data.
    • If exported from the Private (Unlocked) View, it includes IT Comments, Meetings, and Emails.

Step 7: Visual Analytics NEW

Switch instantly between list view and visual charts.

  • How to Access: Click the "üìä Analytics" link in the top navigation bar of the report viewer.
  • What You See: Real-time charts for Project Status, Priority Breakdown, and Workload Distribution.
  • How to Exit: Click "Back to List View" at the bottom of the charts.

Step 8: Sync Outlook (The Agent)

To see your Meetings and Unread Emails, the desktop agent must run.

  1. Open the AquaDailyBriefing shortcut on your desktop.
  2. Wait for the confirmation popup confirming the sync.
  3. Your web report will automatically update with the new data.

Step 9: Viewing Reports

You have two views available based on your needs:

üîí Daily Briefing (Private)

URL: /?daily=YOUR_ID

  • Includes: Tasks, Projects, IT Comments, Meetings, Unread Emails, and Public Comments.
  • Security: Protected by your Passcode. (Press Enter to submit).

üìä Weekly Report (Public)

URL: /?report=YOUR_ID

  • Includes: Tasks, Projects, Resources, and Public Comments.
  • No Sensitive Data: Hides emails, meetings, and IT Comments.
  • Shareable: Perfect for sending to managers or stakeholders as a status update.

Pro-Tip: Automate the Outlook Sync

The Outlook Agent can be configured to run automatically at login.

  1. Navigate to the folder: C:\Users\{username}\AquaBriefing.
  2. Open the configuration file (e.g. dailybriefingconfig.txt).
  3. Set RunAtLogin=Yes.
  4. (Optional) Set Prompt=No to make it run silently in the background without popping up a confirmation box.
